# DungeonGame
Destroy the vampires in the dungeon before your lamp runs out

```
Game options:
  Dungeon(int length, int height, int vampires, int moves, boolean vampiresMove)
  
  length - number of horizontal spaces in dungeon
  height - number of vertical spaces in dungeon
  vampires - number of vampires in dungeon
  moves - number of moves before lamp runs out
  vampiresMove - determines if the vampires move
  
Start game:
  Dungeon dungeon = new Dungeon(10,10,5,20,true);
  dungeon.run();

Keys:
  w - move up
  s - move down
  a - move left
  d - move right
```
